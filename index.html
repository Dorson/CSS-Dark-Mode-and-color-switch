JS Color Theme Switch = CSS Dark Mode for Websites and HTML Apps.

The human day and night cycle is important for our sleep and the eyes. The correct amount of light is very important for good health. During the evening and the night we need dark colors on websites and apps. It is more natural to have the dark background colors at night. This is the live demo page for the open source code that we can use for the Night/Dark Mode = light switch for websites and HTML apps.
